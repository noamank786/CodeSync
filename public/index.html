How It Works

1. Click "Join a Room" to go to the application page.

2. Enter a room ID and username, or create a new room.

3. Start collaborative coding with live updates and chat.

About Code Sync

Code Sync is built to feel like a familiar editor while adding multiplayer powers for remote teams and students.

Technologies Used:

  • Frontend: HTML, CSS, JavaScript, Monaco Editor
  • Backend: WebSockets, real-time collaboration server
  • Realtime Communication: Socket-based messaging
